## Integration Points

### 1. Class Hierarchy

```
SeatSelectionManager (seat-selection.js)
    ↓
DesktopSeatManager (desktop-seat-selection.js)
    ↓
ZoomSeatManager (zoom-seat-manager.js) ← HIGH-PERFORMANCE ZOOM ENGINE
```

### 2. Initialization Order (in tickets-vegas.html)

```html
<!-- 1. Base seat selection -->
<script src="seat-selection.js"></script>

<!-- 2. Desktop-specific features -->
<script src="desktop-seat-selection.js"></script>

<!-- 3. High-performance zoom with LoD rendering -->
<script src="zoom-seat-manager.js"></script>

<!-- 4. Manual initialization & control wiring -->
<script>
  initializeZoomSystem();
  // Connects zoom buttons (+ / - / 🏠)
  // Sets up event listeners
  // Verifies manager loaded
</script>
```

### 3. Zoom Control Buttons

In `tickets-vegas.html` (desktop view):

```html
<button id="desktopResetBtn" title="Reset">🏠</button>
<button id="desktopZoomIn" title="Zoom In">+</button>
<button id="desktopZoomOut" title="Zoom Out">−</button>
```

**Behaviors:**
- **🏠 Reset**: Zoom back to 1x
- **+**: Zoom in by 1.2x at viewport center
- **−**: Zoom out by 0.83x
- **Mouse wheel**: Native zoom-to-point
- **Touch pinch**: Native pinch-to-zoom

### 4. Event Flow

```
User zooms (wheel/pinch/click section)
    ↓
ZoomSeatManager updates target state
    ↓
Animation loop (requestAnimationFrame) interpolates
    ↓
SVG transform applied each frame
    ↓
LoD check: scale >= 2.5?
    ├─ Yes → Inject 25 seats
    └─ No  → Clear seats
    ↓
User clicks seat
    ↓
Dispatch 'zoom:seat-selected' event
    ↓
Cart system listens & updates UI
```

## Configuration

### Zoom Parameters (in zoom-seat-manager.js)

```javascript
this.EASE = 0.12;           // Animation smoothness
this.SCALE_EASE = 0.10;     // Scale animation
this.LoD_THRESHOLD = 2.5;   // Inject seats at scale >= 2.5
this.minScale = 1;          // Min zoom
this.maxScale = 4;          // Max zoom
```

### Tuning Tips

- **Smoother animation**: Lower EASE (0.08-0.10)
- **Snappier response**: Higher EASE (0.15-0.20)
- **More seats earlier**: Lower LoD_THRESHOLD to 2.0
- **Fewer seats**: Raise LoD_THRESHOLD to 3.0
